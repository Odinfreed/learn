📡 Day 1 · 环境搭建（4小时极速版）

核心问题：如何在一夜之间搭建一个合法的“渗透靶场”？

🎯 今日目标

  • 安装虚拟化软件（VMware Workstation Pro）
  • 导入Kali Linux 攻击机
  • 导入Metasploitable 2 靶机
  • 配置网络互通（ping通）
  • 执行首次nmap扫描，验证环境

⚙️ 实战步骤

1. 安装VMware（10分钟）

官网下载 VMware Workstation Pro（现免费），安装时选择“以管理员身份运行”。安装后重启。

2. 下载并导入Kali Linux（30分钟）

从 Kali 官方 下载 VMware 镜像（.7z），解压后用VMware打开 .vmx 文件。
启动后登录 kali / kali，打开终端：

sudo passwd root      # 设置root密码
sudo su               # 切换root
apt update            # 更新软件源（可选，可跳过加快速度）

3. 下载并导入Metasploitable 2（30分钟）

从 SourceForge 下载 Metasploitable-2.zip，用7-Zip解压，VMware打开 .vmx。
网络适配器设为 NAT模式（与Kali保持一致）。
启动后登录：msfadmin / msfadmin

4. 网络连通性验证（15分钟）

在Metasploitable中执行 ifconfig 记下IP（如192.168.xx.xx）。
在Kali终端执行：ping 靶机IP，应收到回复。

💡 若ping不通：检查两台虚拟机网络适配器是否都选“NAT模式”；重启VMware网络服务。

5. 第一次扫描（20分钟）

在Kali中执行：

nmap -sV -O 靶机IP

预期看到开放端口：21(vsftpd 2.3.4)，22(OpenSSH)，80(Apache)，445(Samba)，3306(MySQL)等。
保存结果：nmap -sV -O -oA metasploitable_scan 靶机IP

6. 整理笔记（10分钟）

记录Kali IP、靶机IP、扫描结果截图或文本，写入个人笔记。
